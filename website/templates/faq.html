How do I ...?


Create a Trail
  • From your homepage, enter a trail name in the input box labeled Trail Name, and select New Trail, or Create if this is your first trail. Creating a new trail sets it as active.
Check In
  • There are two ways to check in: Online and Remotely. Before using either method, ensure the trail you wish to drop a marker on is set as active. You can change which trail is active by selecting it from the sidebar on your homepage. The active trail name is displayed at the top.
Check In Online

  • Enter in your coordinate into the respective Latitude and Longitude input boxes, add an optional message (300) in the Message input box, and select the Check-In button.

    Auto coordinate retrieval is currently in development.

Check In Remotely

  • Text or Email your Private Key, Coordinates, and an optional Message to CheckIn@BreadcrumbTrail.app

    Private Key
    Find your Private Key in User Settings. It is your initials, #, and 4 numbers. Do not share this with anyone else.

    Coordinates
    Coordinates must be written in order of Latitude and Longitude, and can use compass direction indicators (N/North, S/South, etc.). The map system is only accurate to 4 decimal places.

    Message
    Messages are optional, but must be formated in a specific way to be included. Messages are also limited to 300 characters. See the formatting below:
    message:"Your message here."

    Examples
    • EX#1111 12.3456 -123.4567
    • EX#2222 Lat 12.3456 Lon -123.4567 message:"This is a check-in with a message"
    • EX#3333 12.3456N 123.4567W message:"This is another check-in that works"
    • Private key is EX#4444. My coordinates are 12.3456 north, and 123.4567 west. Here is my note - message:"Here is an example check-in with extra spacing and characters that also works."
